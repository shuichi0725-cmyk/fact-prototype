​FACT (Verified Intelligence Platform) Spec v5.0
​1. サービス概要
​「答えを出すのではなく、考えるきっかけを作る」ための情報インフラ。人間の主観やバイアスを排除し、AIが多元的なソースを分析・整理して提供する。
​2. コアバリュー
​非民主的設計: コメント、いいね、主観的キュレーションを徹底排除。  
​構造的優位性: 「人が人に伝える」SNS構造から脱却し、AIが複数ソースを分析・担保する知識インフラ。  
​情報の透明性: 信頼度スコアリング（S/A/B/Cランク）と一次ソースへの誘導を重視。  
​3. 主要機能・モード
​🔍 検索モード: 自然言語入力に対し、AIが「要約」「時系列」「ソース一覧」「多角的視点」「関連」の5タブで回答。  
​🗄️ 記録モード: 12の大分類に基づき、蓄積された検証済み情報を3階層（大・中・ソース）で回遊。  
​⚠️ 注意喚起UI: 報道バイアスや科学的不一致など5種類を検出し、3段階（ライン/ポップアップ/詳細）で提示。感度は5段階で設定可能。  
​4. 技術アーキテクチャ
​Stack: Next.js, Tailwind CSS, Claude API (Anthropic), Brave Search API, PostgreSQL, Redis。  
​コスト最適化: * ユーザー自身のClaudeアカウント連携によるAPI費用削減。  
​Embedding類似度検索（0.92以上）によるセマンティックキャッシュ活用。  
​オプトイン制のAI考察共有によるDB蓄積（コストゼロアーキテクチャ）。  
​5. マネタイズ
​収益源: 画面遷移型CM（無料版）、月額サブスク（3プラン）、Amazonアソシエイト。  
​広告ポリシー: 利益相反を防ぐため、メディア・政党・製薬・政府等の広告は全面排除。  
​6. 法的・プライバシー方針
​著作権: 「複製」ではなく「案内（索引化）」に徹し、本文引用は行わずAI要約とリンクのみ提供。  
​プライバシー: 検索履歴はローカル保存、クエリは匿名化。広告目的のトラッキングは一切行わない。  
​7. 開発ロードマップ (Phase 1 MVP)
​Claude API / Web Search 連携実装。  
​信頼度スコア算出および注意喚起の自動検出ロジック構築。  
​ブラウザプラグイン（Chrome等）の技術検証。
